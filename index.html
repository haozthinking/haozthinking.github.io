数组的最大间隔问题

发表于 2017-06-27   |  

问题描述

给定整数数组A[0…N-1]，对这N个数排序后的最大间隔，如:1,7,14,9,4,13的最大间隔为4。

  • 排序后，1,4,7,9,13,14,最大间隔为13-9=4。
  • 对原数组排序，然后求后项减前项的最大值即为解。但此方法的时间复杂度较高，要到$O(N^2)$
  • 这里介绍一种借鉴桶排序的解决方法。

问题分析

假定N个数的最大值最小值为max和min,那么这N个数形成N-1个间隔，其最小值为$\frac{max-min}{N-1}$

  • 如果N个数完全均匀分布，则间距全部是$\frac{max-min}{N-1}$;
  • 如果N个数不是均匀分布，间距不均衡，那么最大间隔肯定大于$\frac{max-min}{N-1}$;

将N个数用间距$\frac{max-min}{N-1}$分成N-1个区间，落在同一区间内的不可能有最大值，因此统计后一区间和前一区间的最大值的差即可。

  • 如果没有任何数据落在某区间，则某区间无效，不参与统计。
  • 如果使用N-1个区间会产生统计误差，而使用N个桶则不会有这种问题。

荷兰国旗问题

发表于 2017-06-25   |  

问题描述

现有红、白、蓝三个不同颜色的小球，乱序排列在一起，请重新排列这些小球，使得红白蓝三色的同颜色的球在一起。

即问题转换为给定数组A[0,1,2,….,N-1],元素只能取0、1、2三个值，设计算法，使得数组排列成”00…0011…1122…22”的形式。

算法分析

我们可以定义三个指针，分别是$begin = 0、current = 0、end = N - 1$

  • 如果A[cur] == 2，那么将A[cur]和A[end]交换位置，然后end–，cur不变；
  • 如果A[cur] == 1，那么将cur++，begin不变，end不变
  • 如果A[cur] == 0，则：
    • 如果begin == cur,那么begin++,cur++
    • begin != cur，则A[cur]与A[begin]交换，begin++，cur不变

通过以上的代码就得到了第一版本的荷兰国旗问题。

进一步考虑：

可知在[begin,current]区间内一定没有2，因此A[begin]要么是0，要么是1；如果begin != current 那么A[begin]肯定是1，因此当A[current]为0时 将A[begin]和A[current]交换也可。交换后A[current] == 1，那么cur也可以++。于是就得到了第二版的荷兰国旗问题。

最大连续子数组问题

发表于 2017-06-25   |  

问题描述

给定一个数组A[0,1,…..,n-1],求A的连续子数组，使得该子数组的和最大。

比如给定数组:{1,-2,3,10,-4,7,2,-5}。那么其最大子数组为{3,10,-4,7,2}。得到的最大子数组加和为18。

算法描述

记S[i]为以A[i]为结尾的数组中的最大子数组，那么S[i+1]=max(S[i]+A[i+1],A[i+1])，且S[0]=A[0]。该算法的时间复杂度为$O(n)$。

零子数组

发表于 2017-06-23   |  

题目描述

对于长度为N的数组A,求连续子数组的和的绝对值最接近0的值。

如:

数组A:[1,-2,3,10,-4,7,2,-5],求他所有子数组中,和最接近0的那些值。并返回最接近0的数组的和的绝对值。

算法描述

计算前n项和数组sum。然后再对sum[-1,0,…..N-1]进行排序,之后计算sum相邻元素差的绝对值,绝对值最小即位所求的数。此算法求sum的时间复杂度为$O(N)$，计算数组相邻元素的差值的时间复杂度为$O(N)$，排序的时间复杂度为$O(N\ logN)$，所以该算法的时间复杂度为$O(N\ logN)$。
